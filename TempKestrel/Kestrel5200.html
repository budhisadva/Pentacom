Con su conjunto completo de mediciones ambientales y características especiales el Kestrel 5200 es calificado como el medidor ambiental más compacto y completo para registro de datos en  profesiones como  técnicos de HVAC, construcción y profesionales de la agricultura.

Su registro de datos puede ser transferido a un dispositivo iOS / Android  por comunicación inalámbrica LiNK y la aplicación Kestrel LiNK, o con un dispositivo Windows / Mac con el accesorio Kestrel LiNK Dongle.

CARACTERÍSTICAS:

  • Pantalla de alta resolución
  • Tamaño compacto
  • Funcionamiento con pilas AA
  • Software: Kestrel Link
  • Variables de medición: Temperatura / Humedad Relativa / Velocidad de viento / Presión barométrica / Densidad y flujo de viento / Índice de estres por calor.

  • Ficha técnica: No
