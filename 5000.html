Kestrel 5000 es la estación ambiental de mano para medición y registro de datos como, temperatura, humedad relativa, indice de calor, velocidad de viento entre otras tantas.   Es extraordinariamente potente con un tamaño compacto.

Características:

  • Pantalla de alta resolución
  • Tamaño compacto
  • Funcionamiento con pilas AA
  • Transferencia de datos vía Bluetooth.
Úsado en investigaciónes, expediciones, agricultura y cualquier  profesión o actividad donde el monitoreo de las condiciones sea prioridad. Resistente a caídas, a prueba de polvo, impermeable y capaz de soportar ambientes severos. Su registro de datos puede ser transferido a un dispositivo iOS / Android  por comunicación inalámbrica LiNK y la aplicación Kestrel LiNK, o con un dispositivo Windows / Mac con el accesorio Kestrel LiNK Dongle.
