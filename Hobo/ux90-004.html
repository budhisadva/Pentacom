El registrador de datos UX90-004 registra las condiciones de encendido y apagado del motor dentro de un campo magnético con su sensor interno, o los cierres de contacto seco mecánicos de los dispositivos externos de detección. Este registrador de datos es ideal para el seguimiento del uso y tiempos de funcionamiento de los motores, bombas, compresores y otros equipos.

Aplicaciones:

  • Monitoreo de HVAC
  • Monitoreo en almacenes
  • Monitoreo ambiental
  • Investigación científica y ambiental
  • En instalaciones farmacéuticas
Tamaño ideal para fácil manejo. Compatible con Software HOBOware. Descarga de datos con interfaz USB.
El UX90-004 es un modelo de registrador de datos fabricado por Onset, una empresa especializada en soluciones de monitoreo ambiental. Este dispositivo se utiliza para medir y registrar datos en una variedad de aplicaciones, incluyendo temperatura, humedad y otros parámetros ambientales.
