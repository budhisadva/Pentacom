El UX100-014 es un registrador de temperatura en ambientes interiores con sensores de termopar. Diseñado para una amplia gama de aplicaciones de monitoreo.  El registrador proporciona una flexibilidad excepcional al aceptar sondas termopares tipo J, K, T, E, R, S, B o N ideales para el registro de temperaturas elevadas. Este modelo tiene capacidad para una entrada Termopar.

Aplicaciones:

  • Procesos industriales
  • Madurez del concreto
  • Interiores
  • Fraguado de cemento
Un registrador de datos de termopar de un solo canal para medir y registrar la temperatura en ambientes interiores. Diseñado para una amplia gama de aplicaciones de monitoreo, desde el monitoreo de procesos industriales hasta las pruebas de madurez del concreto, este registrador brinda una flexibilidad excepcional al aceptar sondas de termopar tipo J, K, T, E, R, S, B o N (se venden por separado).
